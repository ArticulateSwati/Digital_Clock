# 🕰️ Digital Clock

A sleek, responsive, and modern Digital Clock web application built using Vanilla JavaScript, HTML5, and CSS3. This project features a beautiful glassmorphism UI design with real-time updates.

---

![Project Screenshot](./Screenshot%202026-03-27%20124320.png)

---

## 🚀 Features

* **Real-time Updates:** Synchronized with your system clock using JavaScript `setInterval`.
* **12-Hour Format:** Includes AM/PM indicators for user-friendly reading.
* **Glassmorphism UI:** High-end design using `backdrop-filter: blur()`, semi-transparent backgrounds, and subtle borders.
* **Fully Responsive:** Centered layout that adapts to various screen sizes.
* **Dynamic Background:** Uses a high-quality fixed background image for a premium look.

## 🛠️ Technologies Used

| Technology | Purpose |
| :--- | :--- |
| **HTML5** | Semantic structure of the clock elements. |
| **CSS3** | Glassmorphism styling, Flexbox positioning, and custom fonts. |
| **JavaScript** | Logic for time calculation, string padding, and DOM manipulation. |
